### **Стартеры**

Spring Boot предоставляет стартовые пакеты (starters) для быстрого подключения зависимостей:

- **spring-boot-starter-web**: Для веб-приложений. Включает Tomcat, Jackson, Spring MVC.
- **spring-boot-starter-data-jpa**: Для работы с базами данных через JPA.
- **spring-boot-starter-thymeleaf**: Для работы с шаблонизатором Thymeleaf.
- **spring-boot-starter-actuator**: Для мониторинга и управления приложением.
